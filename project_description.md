# Scientific Description

This project analyzes ultraviolet radiance observations from the NASA GOLD
(Global-scale Observations of the Limb and Disk) mission.

## Objectives
- Visualize ionospheric radiance maps
- Investigate Equatorial Ionization Anomaly (EIA) dynamics
- Demonstrate automated processing workflows

## Methods
Data are analyzed using Python scientific libraries including NumPy,
SciPy, and Matplotlib. Machine learning examples demonstrate potential
approaches for detecting EIA crest structures.
