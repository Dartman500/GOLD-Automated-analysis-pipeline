# GOLD Ionospheric Radiance Analysis

![Python](https://img.shields.io/badge/Python-3.10-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![Status](https://img.shields.io/badge/Project-Active-success)

Analysis of ultraviolet radiance data from the NASA GOLD mission to study
equatorial ionization anomaly dynamics.

## Features

- Radiance visualization
- Automated analysis pipeline
- Animation generator
- Machine learning EIA crest detection example
- Jupyter demo notebook

## Project Structure

data/ – input datasets  
notebooks/ – analysis notebooks  
src/ – reusable code modules  
results/ – figures and animations  
docs/ – scientific documentation  
site/ – GitHub Pages visualization site  

## Quick Start

Run the automated pipeline:

```
python run_analysis.py
```
